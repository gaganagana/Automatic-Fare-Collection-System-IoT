# Automatic Fare Collection System with Voice Alert

## Project Overview

This project automates bus fare collection using RFID cards and Arduino UNO.

## Features

- RFID Passenger Identification
- Automatic Fare Deduction
- Voice Alert System
- Servo Motor Gate Control
- LCD Display Notifications

## Components Used

- Arduino UNO
- RFID RC522 Module
- Servo Motor
- LCD Display
- DFPlayer Mini
- Speaker
